Plato is waiting for you.

How to play:

This is a code-breaking game.
Match the colours of the code set by Plato (hidden from you) to win.
No blank options are allowed.

When you're done with your selection, click on the EVAL button.
The pegs on the left will indicate your progress.

  • Black means both the position and colour of your choice of code are correct.
  • White means only the colour is correct.
  • A blank result means that you've gotten it very wrong.

If you use up all your turns before you can guess Plato's code correctly, you lose.

All the best!
